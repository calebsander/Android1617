In order to use CV:
1. Add the jar to the build path, e.g. C:\OpenCV\OpenCV_2_4_9\build\java\opencv-249.jar
2. Add the dll to the path. In the run configuration, add -Djava.library.path=C:/OpenCV/OpenCV_2_4_9/build/java/x64

See
https://opencv-java-tutorials.readthedocs.io/en/latest/index.html
